## 🎯 项目简介

本项目设计了一款**三相 FOC 无刷电机驱动板**，用于驱动无刷电机。项目采用 AI 辅助硬件设计方法，使用 Claude Code、Cursor 等 AI 编程工具进行电路分析、参数计算、设计文档撰写等工作。

### 核心功能

| 参数 | 规格 |
|------|------|
| **产品类型** | 三相 FOC 无刷电机驱动板（纯功率级） |
| **驱动架构** | EG2133 三相半桥驱动 + 6×NCE6080K MOSFET |
| **电流采样** | RS624 四运放差分放大（三相 + 母线） |
| **电机额定电压** | 24V（支持 12-48V） |
| **峰值电流** | ≤80A |
| **控制接口** | 6路 PWM 输入（3.3V/5V 兼容） |
| **采样输出** | 4路电流采样输出（I_U/I_V/I_W/I_BUS） |
| **过流保护** | 软件保护（MCU ADC 读取判断） |

## 🤖 AI 辅助硬件设计工作流

本项目展示了 **AI 辅助嵌入式硬件设计** 的完整工作流程：

### 设计工具链

| 工具 | 用途 |
|------|------|
| **Claude Code** | 电路拓扑分析、参数计算、设计文档撰写 |
| **Cursor** | 原理图审查、PCB 布局建议、代码生成 |
| **LCEDA (立创 EDA)** | 原理图绘制、PCB 设计 |
| **KiCad** | 原理图绘制（备选） |
| **Python** | 参数计算脚本、BOM 管理 |

### AI 工作流程

```
需求分析 ──> 器件选型 ──> 原理图设计 ──> 参数计算 ──> PCB 设计 ──> 验证计划
    │           │            │             │            │            │
    ▼           ▼            ▼             ▼            ▼            ▼
Claude      Claude       Cursor       Claude        Cursor       Claude
Code        Code         AI          Code          AI          Code
(文本分析)  (选型建议)   (布局建议)   (公式推导)    (DRC检查)    (测试计划)
```

### 具体 AI 应用场景

#### 1. 电路拓扑分析
```
使用 Claude Code 分析 FOC 驱动架构：
- 三相半桥驱动电路设计
- 自举电路工作原理
- 电流采样方案对比（下桥采样 vs 串联采样）
```

#### 2. 参数计算自动化
```
使用 Claude Code 进行参数计算：
- 自举电容选型：Cboot = Qg × N / ΔV
- MOSFET 功耗估算：P = I² × Rds(on)
- 电流采样放大器增益计算
- 散热需求分析
```

#### 3. 设计文档生成
```
使用 Claude Code 撰写设计文档：
- 需求规格说明书
- 原理图连接表
- BOM 物料清单
- 验证测试计划
```

#### 4. PCB 布局审查
```
使用 Cursor AI 进行 PCB 审查：
- 走线宽度检查（电流密度）
- 去耦电容布局检查
- 热管理评估
- 信号完整性分析
```

### 关键器件

| 器件 | 型号 | 关键参数 | 封装 | 用途 |
|------|------|----------|------|------|
| 三相半桥驱动 | EG2133 | 3路独立半桥，VCC 4.5-20V，IO +1.2A/-1.4A | TSSOP20 | 栅极驱动 |
| N-MOSFET | NCE6080K | 60V/80A，Rds(on) <8.5mΩ@VGS=10V | TO-252-2L | 功率开关 |
| 四运放 | RS624 | 7MHz GBP，Rail-to-Rail，2.5-5.5V供电 | SOIC-14 | 电流采样 |

### 电流采样方案

采用**下桥采样电阻 + 差分放大**方案：

| 采样通道 | 运放 | 增益 | 范围 | 中点电压 |
|----------|------|------|------|----------|
| I_U / I_V / I_W | RS624 A/B/C | 10× | ±20A | 2.5V |
| I_BUS | RS624 D | 5× | 0~80A | 2.5V |
