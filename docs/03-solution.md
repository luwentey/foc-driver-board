# FOC 驱动板 — 完整方案

> 项目：FOC 三相无刷电机驱动板 | 日期：2026-05-07 | 阶段：完整方案输出（v2 — 需求确认后更新）

---

## 1. 方案摘要

- **产品目标**：三相 FOC 无刷电机纯功率驱动板，外部 MCU 输入 PWM 信号
- **推荐架构**：EG2133 三相半桥驱动 + 6×NCE6080K 功率 MOSFET + RS624 四运放电流采样
- **推荐理由**：用户指定器件，EG2133 集成三路半桥驱动+死区+闭锁，NCE6080K 低 Rds(on) 适合大电流，RS624 四运放全覆盖三相电流+母线电流采样
- **主要风险**：NCE6080K TO-252 封装散热能力限制持续电流；软件过流保护响应依赖 MCU 中断延迟
- **下一步动作**：PCB 布局确认、散热方案评估

#### 自举电路（每相一致）

```
        VCC_12V
           │
        FR107 (或 1N4148，恢复快)
           │  阳极→阴极
           ├──────── VBx
           │        │
         1μF/25V   │
        陶瓷电容    │
           │        │
          VSx ──────┘
```

**自举电容选型**：Cboot = Qg × N / ΔV = 90nC × 1 / (VCC - 2V) ≈ 1μF（留足余量）
- 每个开关周期自举电容需要补充高端 MOSFET 栅极电荷
- FR107 快恢复二极管用于自举（典型应用电路推荐），1N4148 也可用于低压场景

#### 栅极驱动电阻

```
HOx/LOx ─── Rg(10Ω) ──┬── MOSFET Gate
                        │
                      Rgs(10kΩ)    ← 栅源下拉，关态时确保关断
                        │
                    MOSFET Source
```

- Rg = 10Ω：权衡开关速度与 EMI，NCE6080K Qg=90nC，10Ω 下 tr≈8.5ns
- Rgs = 10kΩ：栅源间下拉电阻，确保驱动输出悬空时 MOSFET 关断

### 6.2 NCE6080K 功率 MOSFET 电路

#### 三相逆变桥拓扑

```
               VMOTOR(24V)
                 │
          ┌──────┼──────┐
          │      │      │
         Q1     Q3     Q5       ← 高端 MOSFET (NCE6080K)
         │      │      │
    U_OUT┤ V_OUT┤ W_OUT┤        ← 三相输出节点
          │      │      │
         Q2     Q4     Q6       ← 低端 MOSFET (NCE6080K)
          │      │      │
          └──────┼──────┘
                 │
            Rs_U  Rs_V  Rs_W     ← 相电流采样电阻 (5mΩ)
                 │
               GND_PWR ── Rs_BUS ── 供电地 (母线电流采样)
```

**关键参数验证（24V 额定）**：
- VDS = 60V → 24V 额定电压下有 2.5 倍余量，适合 24V 电机系统
- Rds(on) = 7mΩ(typ) @ VGS=10V → 20A 时导通损耗 P = I²×R = 400×0.007 = 2.8W/管
- Qg = 90nC → EG2133 Io=1.2A 下开关时间 ~75ns，可支持 >100kHz PWM
- 体二极管 VSD = 1.2V @ 20A → 死区续流损耗需评估
- trr = 32ns → 体二极管反向恢复较快，适合 FOC 场景

### 6.3 RS624 电流采样电路

#### 采样方案：下桥采样电阻 + 差分放大

FOC 驱动最常用方案：在下桥 MOSFET 源极到 GND 之间放置采样电阻，检测下桥导通期间的相电流。母线电流通过 DC 回路采样电阻检测。

#### 采样电阻选型

**相电流采样电阻**：
- 阻值：5mΩ
- 假设最大电流 20A，5V 供电，VREF = 2.5V，差分增益 10 倍
- 20A × 5mΩ × 10 = 1.0V → 输出 = 2.5V ± 1.0V = 1.5V ~ 3.5V（5V ADC 量程内）
- 功耗：20A² × 5mΩ = 2W → **选 5mΩ/2W 2512 封装合金电阻**

**母线电流采样电阻**：
- 阻值：5mΩ（与相电流统一，简化 BOM）
- 增益 5 倍 → 40A × 5mΩ × 5 = 1.0V → 输出 = 2.5V + 1.0V = 3.5V
- 80A × 5mΩ × 5 = 2.0V → 输出 = 2.5V + 2.0V = 4.5V（5V 量程内）
- 功耗：40A² × 5mΩ = 8W → **选 5mΩ/5W 3637 封装合金电阻**（或 2×5mΩ/2W 并联）

#### 相电流差分放大电路（每相一致，使用 RS624 的运放 A/B/C）

```
              Vs_U (采样电阻高端)
                │
            R1(1kΩ)
                │
                ├─────── R3(10kΩ) ──┐
                │                    │
            R2(1kΩ)              OP_A (RS624 运放A)
                │                    │  增益 = R3/R1 = 10
              GND_U                 ├─── I_U 输出
                │                    │
                └─────── R4(10kΩ) ──┘
                       │
                   VREF_2.5V
                   (ADC中点)
```

**增益计算（5V 供电）**：
- 差分增益 G = R3/R1 = 10kΩ/1kΩ = 10
- 输出范围：Vout = VREF + (Vs_U - GND_U) × G
- 20A 时：Vout = 2.5V + 20A × 5mΩ × 10 = 2.5V + 1.0V = 3.5V
- -20A 时：Vout = 2.5V - 1.0V = 1.5V

#### 母线电流采样电路（使用 RS624 运放 D）

```
              VMOTOR_RETURN (Rs_BUS 高端，接 GND_PWR)
                │
            R33(2kΩ)
                │
                ├─────── R35(10kΩ) ──┐
                │                    │
            R34(2kΩ)              OP_D (RS624 运放D)
                │                    │  增益 = R35/R33 = 5
           SYSTEM_GND              ├─── I_BUS 输出
                │                    │
                └─────── R36(10kΩ) ──┘
                       │
                   VREF_2.5V
```

**母线采样原理**：
- Rs_BUS 放置在 GND_PWR 与系统地（外部电源地）之间
- 母线电流始终为正（电源 → 板 → 电机），输出 ≥ 2.5V
- 增益 = 5（R35/R33 = 10kΩ/2kΩ），降低增益避免大电流时输出饱和
- 0A 时输出 = 2.5V（VREF 中点）
- 40A 时输出 = 2.5V + 40A × 5mΩ × 5 = 2.5V + 1.0V = 3.5V
- 80A 峰值时输出 = 2.5V + 2.0V = 4.5V（在 5V 量程内）

**RS624 引脚连接（SOIC-14）**：

| 引脚 | 名称 | 连接 | 说明 |
|------|------|------|------|
| 1 | OUT A | I_U 输出 → J2-1 | U相电流采样输出 |
| 2 | -IN A | R3/R4 分压 (U相差分-) | U相反相输入 |
| 3 | +IN A | R1/R3 节点 (U相差分+) | U相同相输入 |
| 4 | V- | GND | 运放负电源 |
| 5 | +IN B | R1/R3 节点 (V相差分+) | V相同相输入 |
| 6 | -IN B | R3/R4 分压 (V相差分-) | V相反相输入 |
| 7 | OUT B | I_V 输出 → J2-2 | V相电流采样输出 |
| 8 | OUT C | I_W 输出 → J2-3 | W相电流采样输出 |
| 9 | -IN C | R3/R4 分压 (W相差分-) | W相反相输入 |
| 10 | +IN C | R1/R3 节点 (W相差分+) | W相同相输入 |

## 7. 软件过流保护方案

### 7.1 工作原理

MCU 通过 ADC 实时读取 4 路电流采样值（I_U、I_V、I_W、I_BUS），与预设阈值比较：

```
ADC 采样 ──> 电流计算 ──> 阈值比较 ──> 超限？──是──> 关断所有 PWM 输出
                │                          │
                │                          否
                │                          │
                └──────────────────── 继续运行 ←┘
```

### 7.2 过流阈值设计

| 参数 | 阈值 | ADC 值(5V参考) | 说明 |
|------|------|----------------|------|
| 相电流过流 | ±30A | I_x > 4.0V 或 < 1.0V | 正常范围 ±20A，1.5 倍余量触发 |
| 母线电流过流 | 60A | I_BUS > 4.0V | 持续超 60A 触发 |
| 相电流短路 | ±50A | I_x > 4.75V 或 < 0.25V | 立即关断 |
| 母线电流短路 | 80A | I_BUS > 4.5V | 立即关断 |

### 7.3 软件实现要求

- ADC 采样率 ≥ PWM 频率的 2 倍（推荐 40kHz+，对应 20kHz PWM）
- 过流判断在中断中执行，响应时间 < 10μs
- 关断方式：将所有 HIN 拉低、所有 LIN 拉高（EG2133 安全态）
- 故障后需软件复位才能恢复，防止自动重启

## 8. PCB 与结构约束

### 8.1 层数建议

**4 层板**，层叠结构：
- L1：信号 + 小功率元件
- L2：GND 完整地平面
- L3：功率地 + 大电流走线
- L4：VMOTOR 电源平面 + 输出走线

### 8.2 布局分区

```
┌────────────────────────────────────────────────┐
│  ┌──────────────┐   ┌──────────────────────┐    │
│  │  控制信号区   │   │    驱动电路区         │    │
│  │  J1 连接器    │   │  EG2133 + 自举电路    │    │
│  │  RS624       │   │  Rg, Rgs 电阻        │    │
│  │  采样电路     │   │                       │    │
│  └──────────────┘   └──────────────────────┘    │
│  ┌──────────────────────────────────────────┐   │
│  │           功率区                          │   │
│  │   Q1  Q3  Q5   (高端 MOSFET)             │   │
│  │   U   V   W    (输出焊盘)                │   │
│  │   Q2  Q4  Q6   (低端 MOSFET)             │   │
│  │   Rs_U Rs_V Rs_W (相电流采样电阻)        │   │
│  │   Rs_BUS        (母线电流采样电阻)       │   │
│  │   VMOTOR ─── GND_PWR (大铜皮)            │   │
│  └──────────────────────────────────────────┘   │
└────────────────────────────────────────────────┘
```

### 8.3 关键布局约束

- **DCDC 热回路**：每个半桥的 VMOTOR → 高端 MOSFET → 输出 → 低端 MOSFET → GND 路径必须最短
- **自举电容**：Cboot 尽可能靠近 VBx-VSx 引脚，走线 < 5mm
- **栅极驱动**：Rg 电阻靠近 MOSFET 栅极放置，栅极走线短而直
- **采样电阻**：相电流采样电阻靠近低端 MOSFET 源极，差分走线对称等长到 RS624
- **母线采样电阻**：Rs_BUS 靠近电源地入口，差分走线到 RS624 运放 D
- **模拟/数字隔离**：RS624 采样区与功率区保持距离，地平面分割在单点连接
- **散热**：NCE6080K TO-252 封装，漏极焊盘需大面积铜皮散热，多打过孔到底层

### 8.4 过孔与走线

| 网络 | 电流 | 最小线宽 | 过孔 |
|------|------|----------|------|
| VMOTOR | ≤80A | ≥5mm (内层铜皮) | 多个 0.8mm 过孔 |
| 相输出 | ≤80A 峰值 | ≥3mm | 多个 0.6mm 过孔 |
| GND_PWR | ≤80A | 铜皮填充 | 多个过孔 |
| 栅极驱动 | ~1A 脉冲 | 0.3mm | 单过孔 |
| 信号 | mA 级 | 0.15mm | 单过孔 |
