## 3. 决策门

### 已确定

- 三相逆变桥拓扑：6×NCE6080K 组成三相全桥
- 驱动方案：EG2133 集成三路半桥驱动
- 电机额定电压：24V
- 电流采样：下桥采样电阻 + RS624 差分放大
  - 运放 A/B/C：三相电流采样，增益 10×
  - 运放 D：母线电流采样，增益 5×
- RS624 供电：5V，VREF = 2.5V（板上电阻分压）
- 自举方案：FR107 + 1μF 陶瓷电容
- 过流保护：软件方式，MCU ADC 读取 4 路电流值判断
- PCB：4层板，65×50mm
- 接口：6路 PWM 输入 + 4路电流采样输出 + 电源接口 + 电机输出
- 散热方案：自然对流，不加散热片
- 连接器选型：5.08mm 插拔端子（电源/电机）+ 2.54mm 排针（信号/采样）
- 母线采样电阻：Rs_BUS 5mΩ/5W 3637 封装单电阻

### 待确认

（无）

### 高风险

- **软件过流保护响应时间**：完全依赖 MCU 中断响应，最坏情况可能 > 10μs。若 MCU 使用硬件比较器触发外部中断可缩短至 < 5μs
- **NCE6080K 无散热片持续电流能力**：TO-252 封装 RθCA ≈ 40°C/W，自然对流下 24V/20A 持续运行壳温将超过 100°C，建议限制持续电流 ≤15A 或缩短满载运行时间
- **母线采样电阻 Rs_BUS 热设计**：5W 功耗在 3637 封装下需充分铜皮散热，自然对流下温升较高
