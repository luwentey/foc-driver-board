## 2. 电源树详细设计

### 2.1 电源域汇总

| 电源域 | 电压 | 来源 | 电流 | 去耦 | 负载 |
|--------|------|------|------|------|------|
| VMOTOR | 24V额定(12-48V) | 外部 | ≤80A峰值 | 100μF电解+1μF陶瓷×3 | 功率MOSFET |
| VCC_12V | 12V±2V | 外部 | ≤500mA | 10μF+0.1μF | EG2133 + 栅极驱动 |
| VB1/2/3 | VCC-1.5V~VCC | 自举 | ~100μA+脉冲 | 1μF/25V陶瓷×3 | 高端驱动 |
| VCC_5V | 5V | 外部 | ≤5mA | 10μF+0.1μF | RS624 |
| VREF | 2.5V (VCC_5V/2) | 电阻分压 | ≤1mA | 0.1μF | ADC参考 |

### 2.2 自举电路设计

每相自举电路完全相同，以U相为例：

```
VCC_12V ──── FR107 阳极
                  │
             FR107 阴极 ──── VB1
                                │
                           Cboot(1μF/25V 陶瓷)
                                │
                            VS1 (U相输出节点)
```

**自举电容计算**：
- Qg_total = Qg_MOSFET + Qc_boot_leakage ≈ 90nC + 10nC = 100nC
- ΔV = VCC - Vf_diode - Vboot_min = 12V - 1V - 0.5V = 10.5V
- Cboot_min = Qg_total / ΔV = 100nC / 10.5V ≈ 10nF
- 选用 1μF/25V X7R 0805 陶瓷电容（100倍余量，确保低压差启动可靠）

**自举二极管选择**：
- FR107：1A/1000V 快恢复二极管，trr ≤ 500ns，典型应用电路推荐
- 替代：US1M（1A/1000V，trr ≈ 75ns，更优）或 1N4148（低压场景 ≤30V）

### 2.3 上电时序

1. VCC_12V 先上电 → EG2133 内部逻辑初始化
2. VCC_5V 上电 → RS624 运放使能
3. VMOTOR 上电 → 功率级就绪
4. MCU 开始输出 PWM → 电机运行

**安全约束**：VCC_12V 必须先于 VMOTOR 上电，确保 EG2133 在功率级带电前已初始化，LIN 上拉确保 MOSFET 关断。

## 4. PCB 约束

### 4.1 板框

- 建议尺寸：65mm × 50mm（4层板，增加母线采样电阻空间）
- 4 × Φ3.2mm 安装孔，位于四角

### 4.2 走线规则

| 网络 | 最小线宽 | 层 | 过孔 | 备注 |
|------|----------|-----|------|------|
| VMOTOR | ≥5mm 铜皮 | L3/L4 | ≥4×0.8mm | 汇流排 |
| U/V/W_OUT | ≥3mm | L1/L4 | ≥3×0.6mm | 短而直 |
| GND_PWR | 铜皮填充 | L2/L3 | 多个 | 完整平面 |
| HO/LO → Gate | 0.3mm | L1 | 1 | 短走线 |
| 采样差分对 | 0.15mm 差分 | L1 | 对称 | 等长等距 |
| HIN/LIN 输入 | 0.15mm | L1 | 1 | 普通 |

### 4.3 阻抗与 EMC

- 功率走线不做阻抗控制，优先短和宽
- 栅极走线远离相输出和 VMOTOR 走线，避免 dV/dt 耦合
- 采样差分对紧耦合走线，远离功率区
- 4层板 L2 完整 GND 平面，提供低阻抗回流路径

### 4.4 散热设计

- NCE6080K TO-252 漏极焊盘 → 大面积铜皮（≥10mm×10mm/管）
- 铜皮通过多个 0.5mm 过孔连接到底层铜皮
- 24V/20A 持续导通损耗 ~2.8W/管，自然对流 RθCA ≈ 40°C/W → ΔT ≈ 112°C
- 建议底部附加散热片或导热垫
- Rs_BUS (5mΩ/5W) 需要铜皮散热，远离信号区
